#  Sales performance powerBI

## Project Overview
Built an interactive Power BI dashboard using the Northwind dataset.

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling

## Data Model
Star Schema Design

### Fact Table
- Order Details

### Dimension Tables
- Customers
- Products
- Categories
- Employees
- Shippers
- Date Table

## KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Order Value

## Dashboard Features
- Sales Trend
- Top Products
- Sales by Country
- Sales by Category
- Dynamic Filters

## Key Insights
- USA generated the highest revenue.
- Beverages category contributed the largest share of sales.
- A small number of products generated a large percentage of revenue.
